Base Function: \(y = a*log_b(cx)+d\)



The "Addition" tab allows you to change the value of variable \(d\)




The "Variable" tab allows you to change the value of variable \(c\)




The "Constant" tab allows you to change the value of variable \(a\)




The "Exponent" tab allows you to change the value of variable \(b\), which starts at 10 and goes down.
